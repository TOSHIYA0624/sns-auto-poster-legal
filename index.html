Schedule once. Publish everywhere.

SNS Auto Poster is a scheduling tool that automatically publishes a creator's own videos to their own social media accounts — TikTok, YouTube, Facebook and Instagram — based on a simple schedule kept in a private Google Sheet.

See how it works

Features

Everything needed to run a consistent posting schedule without manual uploads.

Spreadsheet-driven

Manage every post — video, caption, and publish time — in one private Google Sheet. No dashboard to learn.

Scheduled publishing

Set a date and time for each video. The tool publishes it at the right moment, then records the resulting post URL back to the sheet.

Multi-platform

Publish the same short video to TikTok, YouTube Shorts, Facebook Reels and Instagram Reels from a single schedule.

Safe by design

Status tracking (pending → processing → posted) prevents duplicate uploads. Each platform is handled independently.

Captions from one source

Captions are built from the video's title and description, so the same message is used consistently across platforms.

Your accounts only

The tool posts solely to the owner's own connected accounts. There are no public end users and no third-party content.

How it works

From a row in a spreadsheet to a published post.

  1. Add a video to the schedule

    The creator stores a video in their own Google Drive and adds a row to the Google Sheet with the title, description and publish time.

  2. Connect the account

    The creator authorizes the tool to their own social account using the platform's official login (for TikTok, via TikTok Login Kit). Access can be revoked at any time.

  3. The tool runs on schedule

    At each scheduled time the tool downloads the video and uploads it through the platform's official publishing API.

  4. The post goes live

    The video is published to the creator's own account and the resulting post URL is written back to the spreadsheet.

TikTok integration

How SNS Auto Poster uses the TikTok Content Posting API.

SNS Auto Poster integrates with TikTok using TikTok Login Kit and the Content Posting API (Direct Post) so a creator can schedule their own videos to their own TikTok account:

  • Login Kit (user.info.basic) — the creator signs in with TikTok and authorizes the app; the app reads only their basic profile to confirm the connected account.
  • Content Posting API (video.publish) — the app uploads the creator's selected video and publishes it directly to their own TikTok account at the scheduled time, honoring the creator's chosen privacy level.

The app only ever posts content the creator supplies, only to the creator's own account, and only when the creator has scheduled it.
